# LeetCode Top Interview 150 — A C++ Study Guide

A calm, focused reading companion for all 150 problems of LeetCode's *Top Interview 150* list, written in C++ for people learning data structures and algorithms from scratch.

Each problem: plain-English statement → worked example → brute force (with complexity) → optimized solution and the named pattern behind it → interview notes and follow-ups. Plus a C++/DSA primer, a Big-O and STL reference, a pattern cheat sheet, an interview-day playbook, and an 8-week study plan.

**Read it:** open the published page (GitHub Pages) — single self-contained file, works offline too.
